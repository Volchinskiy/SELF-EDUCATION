CHROM
Ctrl + t              -  открыть новую вкладку
Ctrl + Shift + B      -  Show or hide the Bookmarks bar
Ctrl + Shift + O      -  Open the Bookmarks Manager
Ctrl + h              -  Open the History page in a new tab
Ctrl + j              -  Open the Downloads page in a new tab
Ctrl + e              -  начать поиск
Ctrl + f              -  Open the Find Bar to search the current page
Ctrl + g              -  Jump to the next match to your Find Bar search
Ctrl + Shift + J, Ctrl + Shift + I   -  Toggle Open Developer Tools
F7                    -  Turn on caret browsing
Alt + d, Ctrl + l     -  фокус на инпут поиска, поисковую строку
Ctrl + Enter          -  если табуляшией получить фокус на ссулку и прожать эти кравиши, то станичка по ссылке откроеться в новой вкладке

ТЕРМИНАЛ
Ctrl + ` 							-  фокус на терминал, открыть новый терминал если вообще сейчас не существует никакого терминала.
Ctrl + Windows + `    -  переключает терминал из режима на весь экран до того состояние что было ранешье и обратно.
Ctrl + Shift + `      -  открыть новый терминал.
Ctrl + Shift + 5      -  сплитануть терминал.
Ctrl + Pg Up          -  ходить между группами терминалов
Ctrl + Pg Dn          -  ходить между группами терминалов
Alt + ArrowUp         -  ходить между терминалами в группе
Alt + ArrowDown       -  ходить между терминалами в группе
Ctrl + Space + Space  -  убить терминал на котором находиться фокус

VS CODE
Ctrl + Shift + X      -  открыть расширения
Ctrl + Shift + F      -  открыть поиск
Ctrl + Shift + G      -  открыть изминения гита
Ctrl + Shift + E      -  открыть проводник
Ctrl + ,              -  открыть настройки вс кода UI
Ctrl + b              -  открыть\скрыть боковую панель (там где гит, проводник поиск)
Ctrl + n              -  создать файл при фокусе на дерево проводника
Ctrl + Shift + N      -  создать папку при фокусе на дерево проводника
Ctrl + Shift + =      -  фокус на попап уведомление которое всплывает справа снизу
Ctrl + r              -  показать посление открытые рабочие столы (workbench)
Ctrl + w              -  закрыть окно/файл.
Ctrl + s              -  сохраняет файл.
